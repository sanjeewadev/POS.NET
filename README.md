# POS.NET (C#/.NET Point of Sale System)

A robust, data-driven Point of Sale (POS) system engineered with **C#** and **.NET**. 

While modern retail environments require slick frontends, the backbone of any reliable POS is transactional integrity. This project focuses on the core backend architecture, object-oriented principles, and reliable data persistence required to process sales, manage inventory, and handle retail logic efficiently.

### Tech Stack & Architecture
* **Language:** C#
* **Framework:** .NET 
* **Database:** SQLite (Lightweight, local data persistence without the overhead of a full SQL Server instance).
* **Architecture focus:** Object-Oriented Programming (OOP)

### Key Features
* **Transactional Reliability:** Core C# logic designed to process cart totals, calculate taxes, and finalize transactions securely.
* **Persistent Local Storage:** Integrated SQLite to ensure data is saved locally, allowing the system to operate reliably without requiring a constant cloud connection.
* **Clean Code Structure:** Built using standard .NET project architecture, making it easy to scale, test, or connect to a separate frontend client (like React or WPF) in the future.
